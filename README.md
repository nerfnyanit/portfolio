# Portfolio

フロントエンド設計・UI実装を重視した  
**Next.js 製ポートフォリオサイト**

---

## 概要
フロントエンドエンジニアを目指し制作した、
Next.js / React を用いた個人ポートフォリオサイトです。

UI/UX・コンポーネント設計・保守性を意識し、
**実務を想定した構成・実装**を行っています。

---

## URL
- Portfolio: https://portfolio-nerfnyanits-projects.vercel.app
- GitHub: https://github.com/nerfnyanit/portfolio

---

## 使用技術
- Next.js（App Router）
- React
- TypeScript
- Tailwind CSS
- Vercel

---

## 実装内容
- About / Skills / Projects ページの実装
- スキルカード表示
- カウンター / Todo などのサンプル機能
- 状態管理を含むインタラクティブな UI

---

## 工夫した点
- **表示とロジックを分離**し、保守性を意識した設計
- 再利用性を考慮したコンポーネント構成
- Custom Hook を用いたロジックの共通化
- レスポンシブ対応を考慮した UI 実装

---

## 制作背景・目的
フロントエンド開発において求められる
「設計力」「UI実装力」「継続的な改善姿勢」を
アウトプットとして示すことを目的に制作しました。

単に動くものを作るのではなく、
**実務での可読性・拡張性を意識した実装**を心がけています。
